Installer Center

Sideload & Install the Android App

The Warrior X Android client is pre-cooked and compiled inside Google AI Studio, meaning you have instant access to get the executable payload.

ROUTE A — DIRECT APK FILE

Download APK from Settings HUD

You can directly trigger an APK compiled download. The cloud compiler has pre-packed the APK executable for swift side-loading.

  1. Look at the top-right of the Google AI Studio dashboard.
  2. Reveal the **Settings / Project** catalog.
  3. Click **Download/Generate APK** option.
  4. Install resulting app-debug.apk payload.
ROUTE B — COMPILE LOCALLY

Export ZIP Source & Build Natively

Want to modify security protocols or review performance Kotlin files inside native environments? Get the full source payload.

  1. Tap the header menu inside AI Studio workspace.
  2. Specify **Export Project as ZIP** archive.
  3. Unpack on your desk and access with **Android Studio**.
  4. Run gradle assembleDebug or hit Play to run.
Wait, did you know? You can test the live instance of the app securely inside our streaming Android emulator located in the AI Studio editor right now!

Deploying this website to Vercel

Host this landing site and fully working browser simulation on Vercel under custom domain protocols.

Setup steps mapping:

  1. Push this codebase directly to a public/private GitHub repository.
  2. Access your Vercel Dashboard and specify Add New Project.
  3. Connect your GitHub profile and add this repository.
  4. Leave as default. Vercel will discover vercel.json routing immediately.
  5. Click Deploy and watch the server publish in under 10 seconds!
